# PDP Junior

PDP Junior o'quv markazi uchun statik veb-sayt. Filiallar, kurslar, mentorlar va o'quvchilar natijalari haqida ma'lumot beruvchi bir sahifali (SPA-uslub) sayt.

## Tuzilma

```
index.html        Asosiy sahifa (Filiallar, Kurslar, Mentorlar, Natijalar, Aloqa)
pages/login.html   Kirish sahifasi
css/               Uslublar (style.css, login.css)
js/                Skriptlar (main.js, login.js, chat.js)
images/            Rasmlar
```

## Ishga tushirish

Sayt sof HTML/CSS/JS bo'lgani uchun build kerak emas — istalgan statik server bilan ochish kifoya:

```bash
npx serve .
```

so'ng brauzerda `http://localhost:3000` (yoki serverning ko'rsatgan porti) ni oching.

## Texnologiyalar

- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Inter)
